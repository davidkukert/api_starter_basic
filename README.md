## About <a name = "about"></a>

Initial project for a restful API, built with the Nest framework, containing the users, roles, permissions and auth modules.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
